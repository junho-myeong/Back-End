# 용어 정리 공통

컴퓨테이셔널 씽킹: 현실 세계의 문제를 분석하여 해결책을 찾는 과학적 사고법(큰 문제를 작은 문제로 분해하는게 중요)
컴퓨터 프로그래밍: 설계한 해결책을 컴퓨터 명령어로 작성하는 것(소프트웨어 개발이라고도 한다)
하드웨어 : 물리적인 기계 즉 컴퓨터를 말하고
소프트웨어: 프로그램은 하드웨어와 대비되는 개념이다.
알고르즘: 문제에서 일정한 패턴을 발견하고 패턴을 토대로 문제를 해결하는 절차
코딩: 알고리즘을 코드로 표현하는 행동

프로그래밍이란 코딩이라는 비슷한말이다(단, 프로그래밍은 컴퓨터 명령어로 표현한다는 뜻 즉 컴퓨터에 가까움, 코딩은 프로그래밍 언어로 코드를 작성한다는 뜻 즉 언어 쪽에 가까움)
컴퓨터언어(각언어에 맞게 작성된 언어를 컴퓨터어로 컴파일한다) 와 프로그래밍 언어(java문법에 맞게 작성,또는 c언어 문법에 맞게 작성)는 다르다



## c언어
stdin : 콘솔에 입력하는 방식(standard input)
stdout: 콘솔에 출력하는 방식(standatd out)
