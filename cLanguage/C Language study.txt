## c언어 문법
- ;(세미콜론) : c언어는 구문이 끝날 때 세미콜론을 붙인다, 단 내용이 이어지는 문법은 세미콜론을 붙이지 않는다.
- 주석: 사람만 알아 볼수 잇도록 작성하는 부분,주석은 컴파일러가 처리하지 않음, ( 한줄 주석 // , 여러줄 주석 /*~~~*/)
- {}(중괄호)는 코드에 범위를 나타낸다. (중괄호의 위치는 문법으로 정해지지 않았지만 사람들이 주로 사용하는 방법은 줄의 시작 부분에서 중괄호 사용)

## 디버그
- 버그를 잡는데 도와주는 도구
- 중단점(break point)
- 중단점 삽입/삭제 f9
- 디버깅 시작: f5
- 디버깅 중지: shift+f5
- 프로시저 단위 실행: f10
- 한단계씩 코드실행 : f11
- 컴파일 단축키 : f7

## 변수(값을 담는 공간)
- 문자와 숫자로 변수이름 가능
- 단, 변수 이름은 문자로 시작, 숫자로 시작 할수 없음, 그리고 숫자로만으로 사용할수 없음, 변수명에서 대소문자 구분해서 사용한다
- 단, _로 시작 가능
- 예약어로 변수명을 사용할수 없다.

## 정수 자료형
- 정수는 부호와 크기에 따라 자료형이 달라짐
- 정수 자료형은 부호 + 정수 자료형을 조합하여 사용한다.
- signed(-,0,+), unsigned(0,+) 로 부호가 잇는지 없는지 정해진다, 단 signed는 생략이 가능하다, unsigned로 사용하고 싶을때는 무조건 사용해줘야한다.(단 여기서는 int를 생략할순 잇다)

## 문자 자료형
- signed char = char, 여기서는 signed 생략 가능
- unsigned char(그래서 unsigned는 음수가 없어서 양수에 할당할수 잇는 값의 범위가 늘어 난다)

## 정수 자료형 에서 오버플로우와 언더 플로우
- 저장하는 범위가 넘어가는것을 오버 플로우
- 최솟값 보다 작아지면 언더 플로우 라고 한다

## sizeof() 함수는 변수의 크기를 구하고 싶을때 사용한다.
- size = sizeof num1;
- size = sizeof(num1);
