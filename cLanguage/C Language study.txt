## C언어 개발 환경 설정하기
- Visual Studio 2022를 이용하여 개발 환경 설정하기
  - 1. Visual Studio 2022를 다운로드 받기
  - 2. c++를 사용한 데스크톱 개발 클릭하고 설치하기
  - 3. Visual Studio 열기 해서 개발 설정 Visual c++ 로 체크
  - 4. 새 프로젝트 만들기 에서 언어를 c++로 선택 하고 빈프로젝트 만들기
  - 5. 프로젝트 이름 설정하기
  - 6. 소스 파일에서 파일 만들어서 C++ 파일로 만들면 개발 환경 설정 완료 된다.
